# Table

The Table component provides an easy to use table built on top of the [@nearform/react-table] component, along with the [Material UI] Table component.

## Properties

<PropsTable of={Table} />

For the shape of `rows` and `columns` properties, see [@nearform/react-table].

## Typings Definitions

```ts
export interface TableProps {
  title?: string
  rows?: Array<object>
  columns?: Array<object>

  onDelete?: (selected: object) => void
}
```
