# titus-components
The [titus-components](https://www.npmjs.com/package/@nearform/titus-components) package includes reusable components built on top of existing components provided by open source libraries, enhanced an customized to make them more easily reusable and include additional features.

## [Autocomplete](/autocomplete)
The Autocomplete component provides typical autocompletion features using [Material-UI](https://material-ui.com/) and [downshift](https://github.com/paypal/downshift).

It supports synchronous and asynchronous loading of list items.

## Navigation
The Navigation component provides a typical page layout with a header, a left side drawer and a main content area, all built on top of [Material-UI](https://material-ui.com/) lower-level components.

## Table
The Table component provides an easy to use table built on top of the [@nearform/react-table](https://github.com/nearform/react-table) component, along with the [Material-UI](https://material-ui.com/) Table component.

## Uploader
The Uploader component provides and easy to use way to upload files to a backend by pulling together the [react-dnd](https://github.com/react-dnd/react-dnd) library and [Material-UI](https://material-ui.com/) components.

## Wizard
