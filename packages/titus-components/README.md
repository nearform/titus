# Titus Components

Shareable components ready to be dropped in to any React application.

## Features

* Autocomplete
* List item link
* Navigation
* Table
* Wizard

## Installation

```
npm install --S @nearform/titus-components
```

## Development

```
npm install
```

## Build

```
npm run build
```

## Lint

```
npm run lint

# To use eslint autofix
npm run lint:fix
```
